# 默契问答（朋友互动小游戏）

一个**纯前端 H5** 小游戏：创建者先答题生成邀请链接，朋友打开同一链接答题后自动对比得分。适合在**微信/QQ**聊天里直接打开。

## 运行

### 本地运行（推荐）

由于微信/QQ 内置浏览器对 `file://` 有时限制较多，建议用一个本地静态服务器。

- Python 方式：

```bash
cd docs/friends-mini-game
python -m http.server 5173
```

然后用手机访问同一局域网地址（把电脑 IP 换进去）：

- `http://<你的电脑IP>:5173/`

### 部署到公网（最方便分享）

把 `docs/friends-mini-game/` 作为静态站点目录部署即可（GitHub Pages、Vercel、Netlify、任意对象存储静态托管都行）。

## 隐私说明

- 该小游戏**不需要服务器**
- 创建者答案会被编码到 URL 的 `#mq=...` 片段里（不会随请求发到服务器）
- 页面不会上传你的答案到任何地方
